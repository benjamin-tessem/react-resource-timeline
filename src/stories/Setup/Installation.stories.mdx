# Requirements

React Resource Timeline requires the following peer dependencies:

- [React](https://reactjs.org/) (>=18.0.0)
- [Luxon](https://moment.github.io/luxon/#/) (>=3.0.0) to manage dates
- [react-scroll-sync](https://www.npmjs.com/package/react-scroll-sync) (>=0.11.0) to sync scroll between the header and the body of the Timeline
- [emotion](https://emotion.sh/docs/introduction) (>=11.0.0) to style the Timeline

# Installation

## npm

```bash
npm install react-resource-timeline @emotion/react @emotion/styled luxon react-scroll-sync
```

## yarn

```bash
yarn add react-resource-timeline @emotion/react @emotion/styled luxon react-scroll-sync
```

## pnpm

```bash
pnpm i react-resource-timeline @emotion/react @emotion/styled luxon react-scroll-sync
```
