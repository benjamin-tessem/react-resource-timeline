# Development

## Setup

Install the latest Node LTS version from [nodejs.org](https://nodejs.org/en/) and pnpm from [pnpm.io](https://pnpm.io/).

```sh
pnpm install
```

## Lint

```sh
pnpm lint
```

## Test

```sh
pnpm test
```

## Build

```sh
pnpm build # builds sources at src/ to dist/
```
